                    KIDS GAME PLATFORM - PROJECT DOCUMENTATION
================================================================================

OVERVIEW
--------
A Hebrew-language educational game platform for children ages 3-5. Built with
React + Vite, featuring mobile-responsive design, RTL layout, and gamification
elements like scoring and progress tracking.

TECH STACK
----------
- React 18 with Vite
- React Router DOM for navigation
- Framer Motion for animations
- CSS Modules with CSS Custom Properties
- LocalStorage for persistence

FEATURES
--------
- 2-step onboarding flow (welcome, name input)
- Letter learning categories organized by groups of 5 letters:
  - ××‘×’×“×” (×-×”) + quiz
  - ×•×–×—×˜×™ (×•-×™) + quiz (in progress)
  - More categories to be added as letters are created
- Scoring system: 16/8/4/2/1 points based on attempts
- Cute mascot character with mood states
- Mobile-first responsive design
- Audio infrastructure for voice instructions
- Background music with mute/unmute toggle

LETTER GAMES - LAST QUESTION LOGIC:
    The final question in each letter game should be a MULTI-SELECT asking
    the child to select only the LAST 4 LETTERS learned (not all letters).
    This keeps the game focused and avoids overwhelming children as more
    letters are added. Example: For letter ×•, ask for ×’,×“,×”,×• (not ×-×•).

CATEGORY STRUCTURE
------------------
Letters are organized into groups of 5 (+ quiz):
- ××‘×’×“×” (letters-alef-he): ×, ×‘, ×’, ×“, ×” + quiz-alef-he
- ×•×–×—×˜×™ (letters-vav-yod): ×•, ×–, ×—, ×˜, ×™ + quiz-vav-yod
- ×›×œ×× ×¡ (letters-kaf-samech): ×›, ×œ, ×, × , ×¡ + quiz
- ×¢×¤×¦×§×¨ (letters-ayin-resh): ×¢, ×¤, ×¦, ×§, ×¨ + quiz
- ×©×ª (letters-shin-tav): ×©, ×ª + final quiz

--------------------------------------------------------------------------------

STEP 1: ASK FOR EXAMPLE WORDS (DO THIS FIRST!)
----------------------------------------------
Before creating a new letter game, ask which 4-5 words to use.
Words should be:
- Simple, common words kids ages 3-5 know
- Easy to visualize with images
- Clearly start with the target letter

Example for letter ×— (Chet):
- ×—×ª×•×œ (cat)
- ×—×œ×‘ (milk)
- ×—××•×¨ (donkey)
- ×—×œ×•×Ÿ (window)
- ×—×’×•×¨×” (belt)

--------------------------------------------------------------------------------

STEP 2: CREATE THE GAME FILE
----------------------------
Create: src/data/games/learnX.js (X = letter name in English, e.g., learnChet.js)

Use learnZayin.js as template. Structure:

const learnX = {
  id: 'learn-x',
  categoryId: 'letters-xxx-xxx',  // e.g., 'letters-vav-yod'
  name: 'X',                      // Hebrew letter: ×—
  description: '×œ××“×• ××ª ×”××•×ª X',
  icon: 'X',
  type: 'selection',

  questions: {
    easy: [
      // Q1: Introduction - show letter + example words (isIntroMultiSelect: true)
      // Q2: Optional - show different writing styles (if letter has variants)
      // Q3: Multi-select - find all instances of letter in 3x3 grid
      // Q4: Find letter among 6 random letters
      // Q5: Which word starts with this letter? (4 word options)
      // Q6: Find letter among different 6 letters
      // Q7: Another "which word starts with..." question
      // Q8: Multi-select - mark LAST 4 LETTERS learned (e.g., for ×˜: ×•,×–,×—,×˜)
    ],
  },
};

IMPORTANT: The final question (Q8) should ask to select the LAST 4 LETTERS
learned (not all letters from ×). This keeps the game focused.

--------------------------------------------------------------------------------

STEP 5: REQUEST IMAGES (DO THIS LAST!)
--------------------------------------
After the game is built and working, request images for the example words.
Images go in: public/images/

Image requirements:
- Simple, clear illustrations (child-friendly style)
- Transparent or white background
- ~200x200 px recommended
- Filename format: word-WORDNAME.png (e.g., word-chatul.png)

Example request format:
"Need images for letter ×— game:
- word-chatul.png (cat/×—×ª×•×œ)
- word-chalav.png (milk/×—×œ×‘)
- word-chamor.png (donkey/×—××•×¨)
- word-chalon.png (window/×—×œ×•×Ÿ)
- word-chagora.png (belt/×—×’×•×¨×”)"

--------------------------------------------------------------------------------

STEP 6: CREATE QUIZ WHEN CATEGORY COMPLETE
------------------------------------------
When all 5 letters of a category are done, create a quiz game.
Use quizAlefHe.js as template.

Quiz structure:
- 15 questions (3 per letter)
- 3 question types per letter:
  1. Find letter X among 9 letters (3x3 grid)
  2. What letter does word start with? (shows word image, 9 letter options)
  3. Which word starts with letter X? (4 word options)

Quiz naming: quiz-X-Y.js (e.g., quiz-vav-yod.js)
