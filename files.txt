OVERVIEW
--------
A Hebrew-language educational game platform for children ages 3-5. Built with
React + Vite, featuring mobile-responsive design, RTL layout, and gamification
elements like scoring and progress tracking.

TECH STACK
----------
- React 18 with Vite
- React Router DOM for navigation
- Framer Motion for animations
- CSS Modules with CSS Custom Properties
- LocalStorage for persistence

FEATURES
--------
- 3-step onboarding flow (welcome, name, age selection)
- 3 game categories: Counting (×¡×¤×™×¨×”), Colors (×¦×‘×¢×™×), Logic (×—×©×™×‘×”)
- Scoring system: 16/8/4/2/1 points based on attempts
- Cute mascot character with mood states
- Mobile-first responsive design
- Audio infrastructure for voice instructions

================================================================================
                              NOTES FOR DEVELOPERS
================================================================================

ADDING NEW GAMES
----------------
1. Create game data file in src/data/games/
2. Export from src/data/games/index.js
3. If new question type needed, create component in src/components/game/
4. Add handling in src/pages/Game.jsx

ADDING AUDIO
------------
1. Place audio files in public/sounds/ (effects) or public/audio/ (instructions)
2. Reference in game data files with URLs like "/audio/filename.mp3"
3. Audio will auto-play on question load via useAudio hook (1 second delay)

GENERATING TTS AUDIO (Hebrew)
-----------------------------
Uses edge-tts library (Microsoft Edge's free neural TTS):

Installation: pip install edge-tts

Generate single file:
  edge-tts --voice "he-IL-HilaNeural" --text "×”×˜×§×¡×˜ ×‘×¢×‘×¨×™×ª" --write-media "public/audio/filename.mp3"

Available Hebrew voices:
  - he-IL-HilaNeural (female, recommended for kids)
  - he-IL-AvriNeural (male)

Optional modifiers for more expressive speech:
  --rate "+10%"    (faster)
  --rate "-10%"    (slower)
  --pitch "+5Hz"   (higher pitch)
  --pitch "-5Hz"   (lower pitch)
